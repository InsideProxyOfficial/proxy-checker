# Proxy Checker

Free tool to check HTTP, HTTPS, SOCKS4, SOCKS5 proxies in bulk.
Real requests — real results.

## Features
- Bulk proxy testing with real HTTP requests
- Speed measurement (ms)
- Country & geolocation detection
- Export live proxies to .txt
- Multi-threaded (up to 50 threads)

## Requirements
- Python 3.7+
- pip install requests

## How to run
```
pip install requests
python app.py
```

## Proxy formats supported
```
ip:port
ip:port:user:pass
http://ip:port
socks5://ip:port
```
